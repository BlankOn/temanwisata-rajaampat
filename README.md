Here is how to run the project

1 Install nodejs

	http://nodejs.org/download/
	
2 Install bower and gulp

type: 
	
    npm install -g bower gulp

3 Install packages

type:

    bower install

4 Build package

type:

    gulp

Then there will be a directory called 'dist' which contains the deliverables.
